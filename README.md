Dagger2 Example
=====================

This is just a sample project to demonstrate way of integrating [Dagger2](https://github.com/google/dagger) into your application.
Additionally it contains also demo of [retrolambda](https://github.com/orfjackal/retrolambda), [lombok](http://projectlombok.org/) and [butterknife](http://jakewharton.github.io/butterknife/) usage.

Feature missing? Pull requests are kindly welcome :)
